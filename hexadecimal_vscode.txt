	1. ASCII
	   American Standard Code 
	   for Information 
	   Interchange
	-> ASCII uses 7bit binary so there are 128 symbols.
	-> IMPORTANT SYMBOLS
		Ã˜ 0  	-> NUL (THIS MEANS NO DATA)
		1-32 	-> random communication codes
		33 - 47 	-> Punctuation
		48 -57 	-> numbers
		58 - 64	-> more punctuation
		65 - 90	-> UPPERCASE ALPHA BETA
		91 - 95	-> even more punctuation
		97 - 122	-> lowercase alphabet
		123-126	-> more punctuation 
		127 	-> DEL (comm code)
